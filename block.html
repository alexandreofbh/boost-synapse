Boost Synapse

block

#include <boost/synapse/block.hpp>

namespace
boost
    {
    namespace
    synapse
        {
        template <class Signal, class Emitter>
        shared_ptr<blocker> block( weak_ptr<Emitter> const & e );
        
        template <class Signal, class Emitter>
        shared_ptr<blocker> block( shared_ptr<Emitter> const & e );
        
        template <class Signal, class Emitter>
        shared_ptr<blocker> block( Emitter * e );
        }
    }

Effects:

Blocks the specified Signal from the emitter e until the returned blocker object expires. While the Signal is blocked, calls to emit<Signal> for e are ignored and return 0.

Warning:

If block is passed a raw pointer, deleting the emitter before the returned blocker object has expired results in undefined behavior.

Notes:

  • Blocking affects existing as well as future connections.
  • If the Signal was not blocked for e at the time block is called, it emits meta::blocked<Signal> from the meta::emitter, passing true for the is_blocked argument;
  • When the returned blocker object expires, if there are no other blocker objects blocking the Signal from the same emitter e, the meta::blocked<Signal> is again emitted from the meta::emitter with false passed for the is_blocked argument;
  • The blocker object is passed to handlers of meta::blocked<Signal>. They can use it to access the emitter e passed to block;
  • The returned blocker object does not own e even if the shared_ptr block overload is used.
