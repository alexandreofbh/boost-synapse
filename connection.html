The connect and translate functions return shared_ptr<connection> that is used to control the lifetime of the connection. As well, connection objects are passed to handlers of the meta::connected signal, which can use the emitter member function template to access the emitter object passed to connect/translate, and the connection::set_user_data/connection::get_user_data member function templates to store auxiliary information in the connection object.
