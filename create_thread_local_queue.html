Boost Synapse

create_thread_local_queue

#include <boost/synapse/thread_local_queue.hpp>

namespace
boost
    {
    namespace
    synapse
        {
        shared_ptr<thread_local_queue>
        create_thread_local_queue();
        }
    }

Returns:

A thread-local object that can be used to queue signals emitted asynchronously from other threads. Use poll to emit the queued signals sinchronously into the calling thread. See Interthread Communication Support.

Note:

While any number of threads can use this function to create their own thread_local_queue, it is invalid to create more than one thread_local_queue object per thread.
