meta::emitter

#include <boost/synapse/connect.hpp>

namespace
boost
    {
    namespace
    synapse
        {
        namespace
        meta
            {
            shared_ptr<void const> emitter();
            }
        }
    }

This function returns the meta emitter. Connecting Signals from this emitter allows user code to take action when a particular signal is connected/disconnected or blocked/unblocked.

Note:

The meta emitter is a singleton. It is created the first time meta::emitter is called.
