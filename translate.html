Effects:

As if:

return connect<OriginalSignal>(original_emitter,boost::bind(&emit<TranslatedSignal>,translated_emitter,_1,_2,...),connection_lifetime)

This function can be used to translate one signal into another signal, as long as their arguments are compatible. For example, if we have a button btn and a dialog dlg, translate<button_on_click,dialog_confirmed>(btn,dlg) can be used to confirm the dialog when the button is clicked.

Note:

The returned connection itself does not count towards the return value of emit<OriginalSignal>; that is, the return value of emit<OriginalSignal> is as if emit<TranslatedSignal> was called instead.
